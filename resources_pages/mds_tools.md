| Tool | Purpose | How to Access |
|-------|------|------|
| Canvas | Gradebook | Log in using your CWL |
| GitHub Enterprise | Course materials and assignment distribution | Log in using your CWL |
| GitHub.com | Assignments for specific courses | Create your own account |
| Gradescope | Assignment submission | Access granted through Canvas\* |
| PrairieLearn | Pre-lecture quizzes and MDS quizzes | Log in using your CWL |
| PrairieTest | Quiz and review session reservations | Log in using your CWL, invitation sent once per term |
| Slack | Primary communication tool | Invited via email |

\*Note: Gradescope username is the same as the Canvas primary email. Do **not** change your primary Canvas email address during the MDS program.
